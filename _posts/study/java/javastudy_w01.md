# 1. JVM이란 무엇인가
JVM은 Java Virtual Machine의 줄임말이다. 단순 직역하면 자바 가상 머신이다. 자바의 유명한 문구 중 `"Write Once, Run Anywhere"` 라는 말이 있다. 특정 운영체제에 종속적이지 않고 하나의 코드로 여러 환경에서 동일하게 실행이 가능하다는 자바만의 특성을 잘 나타내는 말이라고 생각한다.    
하지만 자바 코드 자체가 마법처럼 알아서 컴파일 한 번에 모든 컴퓨터가 알아듣는 0과 1로 이루어진 `바이너리 코드`로 변환되는 것은 아니다. 자바 문법으로 작성된 코드의 실행은 OS가 직접 0과 1로 바꾸는 방식이 아니다. `.java` 소스코드는 자바 컴파일러를 통해 일차적으로 `바이트 코드`로 변환된다.JVM이라는 각 운영체제에 알맞게 개발된 가상 머신이라는 녀석이 1차 가공을 거쳐 `바이트 코드`로 변환한 뒤, 그 변환된 코드를 운영체제에 넘겨주어 0과 1로 다시 변환하는 방식으로 돌아간다.
